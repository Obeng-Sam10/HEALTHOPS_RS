Rust CLI Automation Health IT

HealthOps_rs

A lightweight command-line tool for monitoring health-system logs and network activity. Built with Unix-style text processing concepts in Rust.

The Problem

Small clinics, labs, and student health projects often rely on manual checks of logs and network activity. This is slow and error-prone. HealthOps_rs automates these checks and highlights potential issues early.

Core Features

Log Scanning

  • Counts total log lines
  • Detects error patterns
  • Warns when thresholds are exceeded

Network Monitoring

  • Reads OS network data
  • Counts active connections
  • Flags abnormal behaviour

Developer Workflow

  • Fast CLI execution
  • Simple commands
  • Easy to extend

How to Use


# Show help
cargo run help

# Scan log files
cargo run scan logs

# Monitor network activity
cargo run scan network
